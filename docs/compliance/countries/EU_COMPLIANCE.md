# BookFlow — EU Compliance Profile

## Applicable Law
- GDPR

## Data Handling
- Tenant-isolated storage
- Soft delete by default
- Right-to-access supported
- Right-to-erasure via soft delete

## Transfers
- No cross-tenant sharing
- Infrastructure supports EU-hosted DBs

## Statement
BookFlow’s architecture aligns with GDPR principles and can support EU customers with minimal configuration.
